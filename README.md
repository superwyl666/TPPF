# TPPF
source code for the paper "Guidance for twisted particle filter: a continuous-time perspective" by Jianfeng Lu and  Yuliang Wang (https://arxiv.org/abs/2409.02399) (updated version https://arxiv.org/abs/2512.18598)

For each model, run the .ipynb file.

If not computing the relative variance, change ifRV = 1 to ifRV = 0 in each solver.py as commented.

Parameters like d, T, dt can be adjusted in the first block in each .ipynb file.
